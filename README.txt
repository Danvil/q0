QuestZero (q0) is a template black box optimization library.


Required dependencies:

loki template library
sudo apt-get install libloki-dev

Eigen (version 3) linear algebra library
http://eigen.tuxfamily.org


Installation:

mkdir build
cd build
ccmake ..
make
